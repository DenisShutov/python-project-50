# Gendiff
A program that finds differences between two files.

### Supported Formats:

Input: JSON, YAML/YML
Output: stylish, plain, JSON

### Options:

-f, --format FORMAT - output format (stylish, plain, json), default: stylish
-h, --help - show help message


### Setup
```bash
make install
```
